# 作品说明

作品名称：Letter Rush：星号词阵

Letter Rush 是一款网页端词汇竞猜游戏原型，面向英语学习、课堂互动、社群破冰和轻量派对游戏场景，解决传统词汇训练参与感弱、反馈慢、缺少竞技氛围的问题。

游戏中，玩家进入本地模拟房间后，系统会从主题词库或自定义词库中随机抽取 10 个单词，并将字母隐藏为星号。玩家可以全局开字母，也可以随时抢答完整单词；答对越早得分越高，排行榜实时更新。当前 v0.2.1 支持本地模拟多人、机器人玩家、独立词库文件、自定义词库、操作日志和结算复盘。

AI 主要参与需求拆解、玩法规则设计、页面原型规划、前端代码生成、版本迭代、词库拆分方案和提交材料完善。通过多轮 Prompt 协作，项目从初始想法演进为可直接运行和展示的 Web 交互原型。
